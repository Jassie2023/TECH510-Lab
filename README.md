## How to Run
```bash
git clone https://github.com/Jassie2023/TECH510Lab1.git
cd TECH510Lab1
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
pip install -r requirements.txt
streamlit run app.py
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
